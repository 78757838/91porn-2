# 91porn
突破每天看10个视频限制

1. 请检测python 版本，python -V ,版本>=3.0

     Python 3.4.6

2. git clone https://github.com/depfish/91porn.git

3. 安装依赖
   pip3 install < requirements.txt

4. 执行 python start.py 即可以进行下载

下载完成后保存在 91videos 目录
